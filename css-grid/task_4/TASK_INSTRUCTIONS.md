In the `styles.css` file, locate the `.tech-grid` selector:

- Apply a grid display
- Create a 4 column grid (4 equal-width columns)
- Apply spacing between grid items of `30px`
- Center grid items horizontally within their cells
- Center grid items vertically within their cells

locate the `.tech-item` selector:

- Apply a flex display
- Set the flex direction to column
- Center content vertically using flex justify
- Center content horizontally using flex align
