# HTML Advanced

<br>

In this project, I learned how to build a webpage from scratch based on a designer file (Figma). The main focus here is on the **HTML** structure only — no CSS, no styling — just pure **HTML semantic** markup. The goal is to create a solid and well-structured HTML foundation that will later be styled with CSS in future projects.

<br>

The webpage is a "SmileSchool" website that includes a header with navigation, a hero/banner section, a quote section, a list of popular video tutorials, a membership section, an FAQ, and a footer with social links.

<br>

![SmileSchool preview](https://i.ibb.co/pKGgC0J/Homepage-Desktop.png)

<br>

## Learning Objectives

- How to create an HTML page from a wireframe/design
- Proper use of HTML5 semantic tags (header, main, section, footer, etc.)
- How to structure and organize content with correct heading hierarchy (h1, h2, h3...)
- How to embed images, links, buttons, and blockquotes
